JavaFX CSS Generator
====
This app make JavaFX's CSS easily.

This app use JFoenix. http://jfoenix.com/index.html

# Getting start

```
$ git clone https://github.com/toastkidjp/javafx_css_generator.git
$ gradle clean jar
$ java -jar css_generator.jar
```
